📊 Flipkart Mobile Sales Dashboard

🔍 Project Overview

This project presents an interactive Power BI dashboard built using Flipkart mobile dataset. It provides insights into pricing, brand performance, product distribution, and customer ratings.

🎯 Objectives

Analyze mobile pricing trends
Compare brand-wise performance
Identify top expensive mobile models
Understand the relationship between price and rating
📊 Dashboard Features

🟦 Page 1: Sales Overview

KPI Cards (Total Products, Distinct Brands, Avg Price, Avg Rating)
Brand-wise Product Count
Average Selling Price by Brand
Rating Distribution
Slicers (Brand, Memory, Storage)
🟦 Page 2: Product Analysis

Average Selling Price by Model
Top 10 Most Expensive Models
Price vs Rating (Scatter Chart)
Storage-wise Product Count
Navigation Buttons (Next / Previous)
⚙️ Tools Used

Power BI
MS Excel
